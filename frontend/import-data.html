Import Data from CSV

Upload CSV files to bulk import devices, rooms, or other data

Instructions

CSV Format Requirements:

  • Devices CSV: Must have columns: device_unique_id, type_name, brand_name, model, serial_number, purchase_date, warranty_period, notes
  • Rooms CSV: Must have columns: room_number, room_name, building, floor, capacity
  • Installation History CSV: Must have columns: device_unique_id, room_number, installation_date, installation_time, installer_name, installer_id, withdrawal_date, withdrawal_time, withdrawer_name, withdrawer_id, data_entry_by, status, notes
  • First row must be header row with column names
  • Date format should be: YYYY-MM-DD (e.g., 2025-01-15)
  • Time format should be: HH:MM:SS (e.g., 14:30:00) - Leave empty if not applicable
  • For Installation History: status should be 'active' or 'withdrawn'. Leave withdrawal fields empty for active installations
  • File must be in CSV format (.csv extension)
